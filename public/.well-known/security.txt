# Security Policy

If you discover a security vulnerability in Mossaique, please report it
via GitHub Security Advisories or email.

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)

We take security seriously and will respond to valid reports promptly.

# Scope

In scope:
- XSS vulnerabilities
- Authentication/authorization issues
- Data leakage
- CSRF vulnerabilities
- Any security issue that could affect users

Out of scope:
- Social engineering
- Denial of Service (DoS)
- Issues in third-party services
- Reports from automated scanners without validation

Thank you for helping keep Mossaique secure!
